Help

Note: not all languages supported are shown as autocomplete options while typing commands.

Translate

  • /t to translate an inputted sentence
    1. words (required): the words to be translated.
    2. from: language to translate from (defaults to 'auto').
    3. to: language to translate to (based on whether each is specified, defaults to user default language, then server default language, then 'english').
    4. pub: whether to post this translation publicly (whether this message shows up only for you, or sends as a message into the channel).
  • Context menu on a message to access Apps -> translate
  • React to a message with a Flag Emoji to translate the message to languages used in that country.

Information and Help

  • /i to access default server and user languages
  • /help to see a short list of available commands and a link to the bot page

Default Languages

  • /user to set your own default language (if no options are specified, reverts the server default to English)
    1. lang: the user default language.
    2. rev: whether to revert user default language to English.
  • /server if you have manage server permissions, use this to set the server language
    1. lang: the server default language.
    2. rev: whether to revert server default language to English.
